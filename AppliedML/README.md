Lesion Classification and Emotion Detection
-A university project where we experimented with different machine learning models and evaluated the results.

Skin Lesion Classification (Images)
-eksamenopgave1.ipynb: Code for shallow learners and their results.
- model_*.ipynb: Code for the different CNN models experimented with.

Emotion Detection (Texts)
-Problem2_Q1_*.ipynb: Code for data preprocessing and exploratory data analysis.
- RNN_final.ipynb: Final model used for the emotion detection task.
-BlyeSky3.ipynb: Code for preprocessing and applying the final model on a new dataset.
